# 🎨 Dev Color Toolkit

A sleek, high-performance Chrome extension designed for developers and designers to capture, convert, and manage colors effortlessly. Built with React, Tailwind CSS, and Vite.

## 🚀 Features

* **Color Picker:** Instantly grab any color from your screen using the Eyedropper API.
* **Instant Conversion:** Automatically see HEX, RGB, HSL, and nearest Tailwind CSS color equivalents.
* **Gradient Generator:** One-click generation of beautiful CSS gradients.
* **Recent History:** Never lose a color—the extension keeps track of your recently picked colors in local storage.
* **Clean UI:** Modern, dark-mode, glassmorphism-inspired interface with no scrollbars.

## 🛠️ Tech Stack

* **Framework:** React
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **Icons/UI:** Custom optimized layout for Chrome Popup constraints

## 📥 Installation

1.  **Clone this repository:**
    ```bash
    git clone [https://github.com/rohantastic/dev-color-toolkit.git](https://github.com/rohantastic/dev-color-toolkit.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Build the project:**
    ```bash
    npm run build
    ```
4.  **Load into Chrome:**
    * Open Chrome and navigate to `chrome://extensions/`.
    * Enable **Developer mode** (top right corner).
    * Click **Load unpacked**.
    * Select the `dist` (or `build`) folder from your project directory.

## 💡 Usage
* Click the extension icon to open the popup.
* Click **"🎯 Pick Color"** to start picking colors from any webpage.
* Click **"Copy"** on any value to add it to your clipboard instantly.
